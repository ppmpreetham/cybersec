# OSINT

OSINT stands for Open Source Intelligence. It is the process of collecting data from publicly available sources to be used in an intelligence context.

---

## Why OSINT?

OSINT is a valuable resource for intelligence gathering. It can be used to gather information on a target, such as an individual or organization. This information can be used to identify potential threats, assess vulnerabilities, and develop strategies to mitigate risks.

---

## Types of OSINT

### SOCINT (Social Media Intelligence)

SOCINT involves collecting and analyzing data from social media platforms. This can include information from platforms such as Facebook, Twitter, Instagram, LinkedIn, and others. SOCINT can provide insights into an individual's or organization's activities, connections, and interests.

**Common Techniques:**
- Username Enumeration
- Post Analysis
- Network Mapping
- Sentiment Analysis

**Username Enumeration Tools:**
- [Vesper](https://github.com/ppmpreetham/vesper)
- [Sherlock](https://github.com/sherlock-project/sherlock)
- [Maigret](https://github.com/haakont/maigret)

### GEOINT (Geospatial Intelligence)

GEOINT involves collecting and analyzing geospatial data. This can include satellite imagery, maps, and other geographic information. GEOINT can provide insights into an individual's or organization's location, movements, and activities.

### HUMINT (Human Intelligence)

HUMINT involves collecting and analyzing information from human sources. This can include interviews, observations, and other forms of direct interaction. HUMINT can provide insights into an individual's or organization's intentions, motivations, and capabilities.

### SIGINT (Signals Intelligence)

SIGINT involves collecting and analyzing electronic signals. This can include communications between devices, such as phone calls, emails, and other forms of digital communication. SIGINT can provide insights into an individual's or organization's communications, relationships, and activities.

### Common Google Dorks:

- `allintext:"keyword"` → allintext
- `intext:"keyword"` → intext
- `inurl:"keyword"` → inurl
- `allinurl:"keyword"` → allinurl
- `intitle:"keyword"` → intitle
- `allintitle:"keyword"` → allintitle
- `site:"www.google.com"` → site
- `filetype:"pdf"` → filetype
- `link:"keyword"` → link
- `numrange:321-325` → numrange
- `filetype:pdf & (before:2000-01-01 after:2001-01-01)` → before/after
- `inanchor:rat` → allinanchor / inanchor
- `allinpostauthor:"keyword"` → allinpostauthor / inpostauthor
- `related:www.google.com` → related
- `cache:www.google.com` → cache
